## 初衷

我一直想弄一个个人网站，展示自己的同时，积累自己的开发经验并分享编程实践知识,

这个想法开始于我初学编程（2019 年），当时只是想要做一个酷炫的网站，但是技术不是很好也没有好的设计能力，便将此事给耽搁

直到 2024 年回到深圳工作并打算在这边稳定工作，就把先前的计划重新拾起

于是乎使用自己感兴趣的技术栈（基于 `Next.js`, `TypeScript`, `Tailwind CSS` ）构建这一个网站

## 如何运行

克隆代码

```sh
git clone git@github.com:BlueOrgreen/yunfan-website.git
```

安装依赖

```sh
pnpm install

# 或者
yarn install
```

启动开发服务

```sh
pnpm run dev
# 或者
yarn dev
```

然后修改 `src/contents/` 里面的内容即可。
